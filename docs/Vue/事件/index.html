事件

为元素绑定事件，

  1. 绑定事件使用v-on指令

                v-on:事件名

                @事件名

    2.绑定事件的两种方式

                a.内联事件处理器

                        -事件触发时，直接执行js语句

                b.方法事件处理器（vue帮我们调的）

                        -事件触发时，vue会对事件的函数进行调用

                vue如何区分两种处理器：

                            检查事件的值是否合法的js标识符或属性访问路径，

                                        如果是，则表达它是方法事件处理器    

                                        否则，表示它是内联事件处理器

               两种处理器参数不同

                        方法事件处理器，回调函数的参数由vue传

                        内联事件处理器，回调函数的参数由我们自己传

                                    -参数就是事件对象
<h1>{{count}}</h1>
  <button v-on:click="count++">点我一下</button>
  <hr>
  // 只传了一个函数，这个函数最终是vue帮我们调的 方法事件处理器
  <button @click="click">也点我一下</button>
  <hr>
  //内联时间处理器
  <button @click="click()">也点我一下</button>
</template>

<script setup>
import {ref} from "vue";

const count = ref(0)
function click(){
  alert("函数执行了")
}

js标识符

JavaScript标识符是用于标识变量、函数、对象等的名称。标识符必须遵循以下规则：

  1. 可以包含字母、数字、下划线（_）和美元符号（$）。
  2. 第一个字符必须是字母、下划线或美元符号，不能是数字。
  3. 标识符区分大小写。
  4. 不能使用JavaScript关键字作为标识符。
  5. 不能包含空格或其他特殊字符，如@、#、%等。

例如，以下是有效的JavaScript标识符：

myVariable _MyVariable $myVariable my_variable myVariable123

以下是无效的JavaScript标识符：

123myVariable my Variable myVariable@ function (这是JavaScript关键字)

javaScript事件对象是指在事件触发期间，由浏览器创建的一个包含事件相关信息的对象。当事件发生时，浏览器会自动创建事件对象，并将该对象作为一个参数传递给事件处理函数。

事件对象包含许多有用的属性和方法，例如：

  1. type：事件类型
  2. target：事件对象对应的DOM元素
  3. currentTarget：事件当前接收处理的DOM元素
  4. clientX、clientY、pageX、pageY：事件触发时的坐标信息
  5. keyCode：键盘事件触发时的按键信息
  6. preventDefault()：取消事件的默认行为
  7. stopPropagation()：阻止事件的进一步冒泡或捕获

通过访问事件对象的属性和方法，我们可以获取事件发生的详细信息，并对事件进行适当的处理。

在内联事件处理器中，可以使用$event来访问事件对象

function click(...args){
  alert("函数执行了")
  console.log(args)
}

<button @click="click(1,2,$event)">也点我一下</button>
